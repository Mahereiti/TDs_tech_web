# TDs_tech_web
Projet académique en 4 phases (html, css, js, vue.js) comprenant un outil de monitoring de sécurité simple (TD1) et un catalogue d'articles réactif (TD2)

## TD1 Serveur Sécurité - Monitoring
Une interface de contrôle dédiée à la cybersécurité pour surveiller l'activité d'un serveur et enregistrer des alertes.

### Fonctionnalités : 
- Un tableau de bord listant les intrusions ou scans de ports
- Un formulaire pour configurer de nouvelles règles de sécurité (IP, Port, Sévérité, Action de blocage)
- Une liste de rapports (qui ne renvoie nulle part)

## TD2 : Gestionnaire de Stock - Catalogue
Un catalogue de produits interactif type e-commerce.

### Fonctionnalités : 
- Affichage d'une grille de produits
- Barre de recherche en temps réel
- Formulaire d'ajout d'un article

## Evolution technique en 4 Phases
- Phase 1 : Structure (HTML5)
- Phase 2 : Ajout de la Présentation (CSS3)
- Phase 3 : Ajout Dynamisme (JavaScript Natif)
- Phase 4 : Transformation en Vue.js

### Lancement phase 4 : 
cd TD1_phase4/ProjetVues
